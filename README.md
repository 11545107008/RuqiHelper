# 如祺司机好单助手

## 📱 功能说明

自动监听如祺司机端订单弹窗，识别单价≥1.5元/公里的好单，并通过**悬浮窗+震动**提醒司机手动抢单。

### 核心功能
- ✅ 监听如祺司机端订单弹窗
- ✅ 自动计算每公里单价
- ✅ 好单提醒（悬浮窗绿色弹窗+震动）
- ⚠️ 可选自动点击接单（风险高，可能被封号）

---

## 🛠️ 编译安装

### 方法1：使用Android Studio（推荐）

1. 下载安装 [Android Studio](https://developer.android.com/studio)
2. 打开本项目（RuqiHelper文件夹）
3. 等待Gradle同步完成
4. 连接手机或启动模拟器
5. 点击 ▶️ Run 按钮安装到手机

### 方法2：命令行编译

```bash
cd RuqiHelper
./gradlew assembleDebug
```

生成的APK在：`app/build/outputs/apk/debug/app-debug.apk`

---

## 📲 使用方法

### 第一步：安装APP
将APK安装到手机，首次打开会请求权限。

### 第二步：开启无障碍服务
1. 打开如祺好单助手APP
2. 点击"开启无障碍服务"按钮
3. 在系统设置中找到"如祺好单助手"
4. 开启无障碍服务

### 第三步：开启悬浮窗权限
1. 返回APP
2. 点击"开启悬浮窗权限"按钮
3. 允许显示悬浮窗

### 第四步：打开如祺司机端
1. 打开如祺司机端APP
2. 等待订单弹窗
3. 当检测到单价≥1.5元/公里的好单时：
   - 手机震动
   - 屏幕顶部显示绿色悬浮窗提醒
   - 手动点击接单

2. **自动点击风险**：开启自动点击功能会显著增加被检测风险
3. **UST作者不承担责任**：本工具仅供学习研究，使用后果自负

### 安全建议
- ✅ 只使用提醒功能，手动点击接单
- ✅ 不要频繁使用，避免被检测
- ✅ 定期关闭工具，模拟正常使用情况
- ❌ 不要开启自动点击功能

## 🔧 技术原理

```
如祺司机端订单弹窗
        ↓
AccessibilityService监听
        ↓
遍历界面节点提取文本
        ↓
正则匹配金额和距离
        ↓
计算单价 ≥ 1.5元/km？
        ↓
是 → 悬浮窗提醒 + 震动
```
